# 📘 星際交易所 (Interstellar Exchange) 使用說明書

歡迎使用 **星際交易所**。本系統提供現貨交易模擬功能，具備即時撮合引擎與自動化造市機器人，讓您體驗真實的加密貨幣交易流程。

## 1. 啟動與登入
1.  **進入系統**：開啟瀏覽器，系統會自動導向至登入頁面。
2.  **註冊帳號**：
    * 若您是新使用者，請點擊登入框下方的「申請加入」。
    * 填寫 **帳號**、**密碼**、**姓名** 與 **電話**。
    * 點擊「提交資料」完成註冊。
3.  **登入**：
    * 使用剛註冊的帳號密碼登入。
    * 登入成功後將進入「艦橋主控台 (Dashboard)」。

## 2. 資產管理 (My Wallet)
在主控台點擊「我的錢包」卡片，即可進入資產管理頁面。

* **查看資產**：列表顯示您持有的所有幣種，包含：
    * **總額 (Balance)**：您擁有的總資產。
    * **可用 (Available)**：可自由支配的餘額。
    * **凍結 (Frozen)**：因掛單而被鎖定的資金。
* **模擬儲值**：
    1.  點擊任一幣種卡片上的「儲值」按鈕。
    2.  輸入欲儲值的金額（例如 `10000`）。
    3.  點擊確認，系統即時入帳。
* **一鍵重置**：
    * 點擊右上角的「一鍵重置」按鈕。
    * 確認後，所有幣種的餘額將歸零（用於重新測試）。

## 3. 交易操作 (Trading Center)
在主控台點擊「交易中心」卡片，進入交易介面。

### 3.1 介面概覽
* **K 線圖 (Chart)**：位於上方，顯示價格走勢。支援 `1m`, `15m`, `1h` 等週期切換。
* **下單區 (Order Form)**：位於左側，進行買賣操作。
* **訂單簿 (Order Book)**：位於中間，紅色為賣盤 (Ask)，藍色為買盤 (Bid)。
* **我的委託 (My Orders)**：位於右側，管理您的訂單。

### 3.2 如何下單
1.  **選擇交易對**：在左上角搜尋框輸入幣種（如 `BTC`），選擇 `BTCUSDT`。
2.  **選擇方向**：點擊「買入 (BUY)」或「賣出 (SELL)」。
3.  **選擇類型**：
    * **限價單 (Limit)**：需輸入「價格」與「數量」。當市場價格達到您設定的價格時才會成交。
    * **市價單 (Market)**：僅需輸入「數量」。系統會以當前市場最優價格立即成交。
4.  **提交**：點擊下單按鈕，系統會凍結對應資金並送出訂單。

### 3.3 訂單管理
* **撤單**：在「我的委託」列表中，對於狀態為 `NEW` 的訂單，點擊「撤單」按鈕可取消委託並退回資金。
* **篩選**：使用上方的過濾按鈕（全部/未成交/已成交）來篩選訂單顯示。

## 4. 個人資料 (Profile)
* 在主控台點擊「會員資訊」。
* 您可以查看帳號詳情，或點擊「修改資料」來更新您的姓名、電話或密碼。
